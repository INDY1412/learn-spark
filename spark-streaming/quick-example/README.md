## File Streaming

### 資料產生端
```shell
$ echo "A B C D E F" > wordCount/0001
$ echo "A B C D E F" > wordCount/0002
```

### 資料處理端
```shell
$ spark-submit --class FileStreaming target/scala-2.11/spark-streaming-app_2.11-1.0.jar
-------------------------------------------
Time: 1475328745000 ms
-------------------------------------------
(B,2)
(F,2)
(D,2)

(E,2)
```
